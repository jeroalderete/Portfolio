## Instalacion :warning:

### :running: Ejecutar Local

Clonar el proyecto

```sh
- git clone https://github.com/jeroalderete/Portfolio
```

Ir a la carpeta del proyecto

```bash
  cd Portfolio
```

Instalar dependencias

```bash
  npm install
```

Start server

```bash
  Run Live Server in Vs Code
```
